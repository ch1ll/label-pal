    Timestamps

      How to use this add-on:

      1. Enter a project name (optional).
      2. Add labels with associated time windows.
      3. Select the labels you want to apply.
      4. Click "Record Timestamp" to capture the current video time.
      5. Review your recorded timestamps.
      6. Export your data to JSON or YAML when finished.

      Note: Make sure you're on a page with a video playing when using this add-on.
