    Timestamps

      How to use this add-on:

      1. Enter a project name (optional).
      2. Add labels with associated time windows.
      3. Select the labels you want to apply.
      4. Click "Record Timestamp" to capture the current video time.
      5. Review your recorded timestamps.
      6. Export your data to JSON or YAML when finished.

        7. When exporting to YAML we supply a start and end timestamp -/+ time window seconds.

        When exporting to JSON just supply the explicit timestamp.

      7. To return to sidebar from pop-up, simply close the window and re-open the add on

      If you are playing a video and get:

      "Error: Unable to get current video time. Make sure you're on a page with a video playing."

      Close and re-open the add on. This might happen after initially loading the temporary extension in about:debugging#/runtime/this-firefox
